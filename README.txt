Meu Portfólio de Projetos
Bem-vindo ao meu repositório de projetos! Aqui você encontrará o código-fonte e a documentação do meu portfólio, onde apresento os principais projetos que desenvolvi ao longo da minha carreira.

Sobre o Portfólio
Este portfólio foi criado para demonstrar minhas habilidades em desenvolvimento web, análise de dados, ETL, e criação de dashboards interativos. Cada projeto destacado no site foi cuidadosamente selecionado para refletir minha experiência e conhecimento técnico.

Principais Funcionalidades
Exibição de Projetos: Navegue pelos meus projetos, com descrições detalhadas e links para visualizá-los em ação.
Processo de ETL: Explicação detalhada sobre os processos de extração, transformação e carga de dados usados em meus projetos de análise.
Dashboards Interativos: Acesso a dashboards que fornecem insights valiosos a partir de dados reais.
Design Responsivo: O site foi otimizado para oferecer uma experiência de navegação agradável em dispositivos móveis e desktops.
Tecnologias Utilizadas
HTML/CSS/JavaScript: Para a estruturação, estilo e interatividade do site.
Python: Utilizado para a manipulação e análise de dados.
Power BI: Ferramenta para criação de dashboards interativos.
MySQL: Banco de dados utilizado para armazenar e consultar dados.
Google Cloud Platform (GCP): Infraestrutura de nuvem onde os dados são armazenados e processados.
